Para alojar (o compartir) un repositorio Git en GitHab, siga los siguientes pasos:
paso 1: Registrar una cuenta de GitHub
El primer paso para comenazar con el alojamiento en GitHub es crear una cuenta personal.
Visite la pagina de registro oficial para registrarse.

Paso 2: Crear un repositorio remoto en GitHub
Despues de registrarse, crea un home (un repositorio) en GitHub
para el repositorio Git que quieres compartir.
